# USE Messenger

Простой мессенджер с чатами в реальном времени.

## Быстрый старт

### Windows
```bash
start.bat
```

### Linux/Mac
```bash
./start.sh
```

## Адреса

- Frontend: http://localhost:3000
- Backend: http://localhost:4000

## Структура проекта

```
messenger/
├── backend/          # Go backend (SQLite + WebSocket)
├── frontend/         # React frontend (Vite)
├── start.bat        # Запуск для Windows
└── start.sh         # Запуск для Linux/Mac
```

## Технологии

- Backend: Go + SQLite + WebSocket
- Frontend: React + Vite + Tailwind CSS
