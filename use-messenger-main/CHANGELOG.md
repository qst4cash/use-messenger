### Возможности приложения
- Регистрация и авторизация (JWT)
- Чаты в реальном времени (WebSocket)
- Отправка файлов (изображения, видео, аудио)
- Голосовые сообщения (запись с микрофона)
- Аудио звонки (WebRTC)
- Аватары пользователей
- Плейлист для аудио файлов
- Удаление сообщений
- Анимации (Framer Motion)

### Технологии
- Backend: Go, Gorilla Mux, Gorilla WebSocket, SQLite, JWT, bcrypt
- Frontend: React 18, Vite, Framer Motion, WebRTC
