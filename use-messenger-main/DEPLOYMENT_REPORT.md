```bash
cd /opt/use-messenger

```

## 🚀 Следующие шаги

### Немедленно (для деплоя)
1. **Получить сервер с 2GB+ RAM**
   - VPS от любого провайдера
   - Ubuntu 20.04+ или Debian 10+
   - Открыть порты 80, 443

2. **Настроить SSH доступ**
   ```bash
   ssh-keygen -t rsa -b 4096
   ssh-copy-id root@<SERVER_IP>
   ```

3. **Настроить DNS**
   - Type: A
   - Name: @
   - Value: <SERVER_IP>
   - TTL: 300

4. **Запустить деплой**
   ```bash
   cd C:\USE\deploy
   chmod +x full-deploy.sh
   ./full-deploy.sh <SERVER_IP> usecommunity.online
   ```

5. **Проверить работу**
   - Открыть https://usecommunity.online
   - Зарегистрироваться
   - Отправить сообщение
   - Проверить WebSocket

### После деплоя
1. **Настроить мониторинг** (опционально)
   - Portainer для Docker
   - Grafana для метрик
   - Uptime Robot для проверки доступности

2. **Настроить CI/CD** (опционально)
   - GitHub Actions для автодеплоя
   - Webhook для обновлений

3. **Оптимизация** (опционально)
   - CDN для статики
   - Redis для кэширования
   - PostgreSQL вместо SQLite
